Gem Blocks

You have 3 types of gem blocks of sizes 1 by 1 by 1, 1 by 2 by 1, and 2 by 1 by 1.

Assume you have an infinite number of blocks of each type. You want to make a wall of height K, width 2‚Åø, and depth 1 out of these blocks. The wall should not have any holes in it. The blocks cannot be rotated. In how many ways can the wall be built, modulo 1000000007?

Input:
The first line contains the number of test cases T. T test cases follow. Each test case contains two integers: N and K.

Output:
Output T lines, one for each test case containing the number of ways to build the wall. As the numbers can be very large, output the result modulo 1000000007.

Constraints:
1 less or equal to T less or equal to 100
0 less or equal to N less or equal to 1024
1 less or equal to K less or equal to 5

Make sure that your program is efficient enough to run the worst case input in a reasonable amount of time on a modern device.
